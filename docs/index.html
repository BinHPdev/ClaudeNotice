Hooks 基础概念

Hook 事件类型：
- PreToolUse    — 工具调用前触发（可阻断）
- PostToolUse   — 工具调用后触发
- Notification  — Claude 发送通知时触发
- Stop          — Claude 停止前触发

典型用途：
- 危险操作拦截（阻止 rm -rf、force push）
- 自动运行 lint/format（每次文件修改后）
- 日志记录所有 Claude 操作
- Slack/钉钉通知长任务完成
- 自动保存对话记录

#!/bin/bash
COMMAND=$1
